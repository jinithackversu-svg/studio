# **App Name**: CanteenConnect

## Core Features:

- Menu Browsing: Customers can browse the available menu items with details such as name, description, price, and daily availability.
- Order Placement: Customers can add items to their cart, adjust quantities, and place orders.
- Real-Time Order Tracking: Customers can track their order status in real time through different stages, from payment to pickup, enhancing user experience.
- Payment Options: Offer online payment and cash options based on operator acceptance for flexibility.
- Digital Invoices: Generate digital invoices with QR codes after operator confirmation for customer validation. AI tool determines the structure and incorporates the relevant data of the customer and their order.
- Menu Management: Operators can manage menu data by adding, editing, and deleting items, and toggle daily availability.
- Order Queue Management: Operators can view and manage the order queue in real time, grouped by status to optimize the fulfillment process.

## Style Guidelines:

- Primary color: Fresh Mint (#98FF98), representing freshness and healthy food choices.
- Background color: Light Beige (#F5F5DC), offering a clean and neutral backdrop.
- Accent color: Soft Orange (#FFB347), used for call-to-action buttons and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a blend of modern design and readability for all content.
- Simple, clear icons that visually represent menu categories and actions.
- A clean and responsive layout that is easy to navigate on both mobile and desktop devices, with an operator interface optimized for efficiency and low clutter.
- Subtle animations to provide feedback on interactions, such as adding items to the cart or updating order status.
